# 🧠 Habit Tracker Application

## 📌 Project Description

Habit Tracker is a full-stack web application that helps users track daily habits, monitor progress, maintain streaks, and analyze performance using detailed reports and analytics.

The app encourages consistency through gamification, achievements, challenges, and visual progress tracking.

---

## ✨ Features

- ✅ User Authentication (JWT)
- 📅 Daily Habit Logging
- 🔥 Streak Tracking
- 📊 Weekly & Monthly Reports
- 🏆 Achievements & Badges
- 🎯 Goal Setting
- 🏁 Challenges
- 🌙 Dark Mode
- 📱 Fully Responsive Design
- 🍔 Mobile Sidebar Menu
- 📈 Advanced Analytics

---

## 🛠 Tech Stack Used

### Frontend:
- React (Vite)
- Tailwind CSS
- Axios
- React Router DOM

### Backend:
- Node.js
- Express.js
- Supabase (PostgreSQL)
- JWT Authentication

---

## ⚙ Installation Steps

```bash
git clone <frontend-repo-link>
cd habit-tracker-frontend
npm install
npm run dev
```
